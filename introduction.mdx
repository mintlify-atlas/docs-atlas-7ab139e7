## Overview

CompuTécnicos is a comprehensive e-commerce platform designed specifically for computer hardware retail. Built with PHP and MySQL, it provides a complete solution for managing online computer parts sales with professional inventory management, accounting features, and Colombian legal compliance.

## Key Features

<AccordionGroup>
  <Accordion title="Product Catalog Management" icon="box">
    Complete product management with categories, brands, advanced filtering, image galleries, and featured/sale products. Supports detailed product descriptions and specifications.
  </Accordion>
  
  <Accordion title="Shopping Cart & Checkout" icon="cart-shopping">
    Seamless shopping experience with persistent cart storage, PayPal integration for secure payments, and order tracking with status updates.
  </Accordion>
  
  <Accordion title="Inventory Management" icon="warehouse">
    Professional inventory control with entry/exit tracking, low stock alerts, inventory valuation, and detailed movement history with document support.
  </Accordion>
  
  <Accordion title="Accounting & Compliance" icon="file-invoice">
    Built-in accounting features with Colombian tax compliance (IVA and retención), electronic invoicing support (Alegra/Siigo integration), and PDF invoice generation.
  </Accordion>
  
  <Accordion title="Admin Dashboard" icon="chart-line">
    Comprehensive analytics dashboard with sales KPIs, revenue tracking, inventory reports, and customer insights.
  </Accordion>
  
  <Accordion title="User Management" icon="users">
    Customer accounts with Google OAuth support, profile management, order history, and role-based access control (admin/customer).
  </Accordion>
</AccordionGroup>

## Technology Stack

CompuTécnicos is built with modern web technologies:

- **Backend**: PHP 7.4+ with PDO for database access
- **Database**: MySQL 5.7+ with comprehensive schema
- **Frontend**: HTML5, CSS3, JavaScript with Flowbite components
- **Payment**: PayPal SDK integration
- **Authentication**: Session-based auth + Google OAuth 2.0
- **PDF Generation**: DomPDF library
- **Excel Export**: PhpSpreadsheet
- **Deployment**: Docker + Docker Compose
